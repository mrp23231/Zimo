📋 Пошаговая инструкция

1
Откройте терминал VSCode
Меню: Terminal → New Terminal
2
Перейдите в папку проекта 
cd /Users/nikitabn/Downloads/minimalist-social-network
3
Выйдите из текущего аккаунта Firebase 
firebase logout
4
Войдите под аккаунтом akkgame158@gmail.com
firebase login
Откроется браузер. Авторизуйтесь с аккаунтом akkgame158@gmail.com
5
Убедитесь, что выбран проект zimo-554fd 
firebase use zimo-554fd
6
Деплой правил и индексов 
firebase deploy --only firestore:rules,firestore:indexes --project zimo-554fd
Это займет 1-2 минуты
7
Перезапустите dev-сервер 
npm run dev
8
Очистите кэш браузера и войдите в приложение заново
